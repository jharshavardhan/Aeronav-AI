✈️ AeroNav AI
Intelligent Eco-Optimized Flight Routing System 🌱

AeroNav AI is a data-driven aviation analytics system that optimizes flight routes for reduced fuel consumption and CO₂ emissions while providing interactive visualizations and AI-generated insights.

🚀 Features
✈️ Multi-aircraft flight analysis (B737, A320, B777, A350, etc.)
🧭 Route optimization strategies:
Eco Optimal
Balanced
Minimum Time
Standard
🌍 Great-circle route computation
🌬️ Wind-aware routing simulation
📊 Interactive dashboards (fuel, CO₂, wind, emissions)
🗺️ Global route visualization using Plotly
🤖 AI-generated insights using Gemini API
🗄️ MongoDB Atlas integration for data storage
📁 JSON export for portability
