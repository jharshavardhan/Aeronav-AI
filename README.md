# ✈️ AeroNav AI

AeroNav AI is an eco-optimized flight routing system that analyzes flight paths to reduce fuel consumption and CO₂ emissions using data-driven modeling, visualization, and AI insights.

---

## 📌 Introduction

Aviation contributes significantly to global carbon emissions. Traditional flight planning focuses on time and cost, often ignoring environmental impact.  
AeroNav AI addresses this by providing an intelligent system for eco-efficient flight analysis.

---

## ⚙️ What it does

- Calculates flight distance (great-circle method)
- Estimates fuel consumption and CO₂ emissions
- Compares multiple routing strategies:
  - Eco Optimal
  - Balanced
  - Minimum Time
  - Standard
- Generates optimized routes with waypoints
- Visualizes routes using maps and dashboards
- Generates AI insights using Gemini API
- Stores data in MongoDB Atlas
- Exports results as JSON
